Получение списка предложений - GetOffers(string[] rangeField, string[] rangeValue, bool newEar, string[] sortField, string[] sortOrder, int limit, int selStart)
Аргументы:
rangeField - Список полей для которых осуществляется фильтрация. Допустимые значения: null, или название любого из полей которые возвращает функция
rangeValue - Список значений ко которым происходит фильтрация. Примечание: количество значений должно совпадать с количеством полей. (допускается мета символ * - любое количество любых символов, _ - один любой символ)
newEar - выбирать только те предложения у которых ClientItemCode не null
sortField - Поля по которым может осуществляться сортировка. Допустимые значения: null, или название любого из полей которые возвращает функция
sortOrder - Порядок сортировки для полей из SortField. Допустимые значения: "ASC" - прямой порядок сортировки , "DESC" - обратный порядок сортовки. Примечание: Количество значений может совпадать или быть меньше количества полей для сортировки. Если количество значений меньше то для оставшихся полей будет применен прямой порядок сортировки (ASC). Пример: SortField = {"OrderID", "Junk" ,"Cost"}, SortOrder = {"DESC"} - в этом случае выборка будет отсортированна так же как и в случае если бы  SortOrder = {"DESC", "ASC", "ASC"}
limit - выбрать не более заданного количества записей
selStart - начинать выборку с заданной позиции (позиции нумеруются с 0)
Возвращает: 
OfferId - Идентификатор предложения
PriceCode - Идентификатор прайс листа
FullCode - Идентификатор наименования
ClientItemCode - Идентификатор наименования по ассортиментному прайс листу заказчика
Code - Идентификатор наименования который назначен данному предложению поставщиком
CodeCr - Идентификатор производителя который назначен данному предложению поставщиком
Name - Полное наименование
CrName - Название производителя
Unit - Единица измерения
Volume - Количество в упаковке
Quantity - Остаток на складе поставщика
Note - Примечание
Period - Срок годности
Doc - Документ
Junk - Флаг уценено ли предложение
RequestRatio - заказываемое количество должно делиться без остатка на RequestRatio
MinOrderSum - минимальная сумма заказа для данной позиции
MinOrderCount - минимальное количество заказа
Cost - Цена

Получения списка доступных поставщиков - GetPriceList(string[] firmName)
Аргументы:
firmName - массив строк по которым будет осуществляться фильтрация названий поставщиков (допускается мета символ * - любое количество любых символов, _ - один любой символ)
Возвращает:
PriceCode - Идентификатор прайс листа
PriceName - Название прайс листа
PriceDate - Дата последнего обновления прайс листа
ContactInfo - Контактная информация
OperativeInfo - Оперативная информация
PublicUpCost - Публичная наценка
DisabledByClient - Флаг указывающий отключен ли прайс лист клиентом
FirmCode - Идентификатор поставщика
FirmShortName - Краткое название поставщика
FirmFullName - Полное название поставщика
RegionPhone - Телефон в регионе
Phone - Телефон
Address - Адрес

Получения списка каталожных наименований - GetNamesFromCatalog(string[] name, string[] form, bool offerOnly, int limit, int selStart)
Аргументы:
name - массив строк по которым будет производиться поиск наименований (допускается мета символ * - любое количество любых символов, _ - один любой символ)
form - массив строк по которым будет производиться поиск форм выпуска (допускается мета символ * - любое количество любых символов, _ - один любой символ)
offerOnly - выбирать только те каталожные наименования по которым есть предложения
limit - выбрать не более заданного количества записей
selStart - начинать выборку с заданной позиции (позиции нумеруются с 0)
Возвращает:
FullCode - Идентификатор наименования
CatalogName - Каталожное наименование
CatalogForm - Каталожная форма выпуска


Сформировать заказ по позициям  PostOrder(long[] offerId, Int32[] quantity, string[] message)
Аргументы:
offerId - массив идентификаторов предложений которые нужно заказать
quantity - количество которое нужно заказать
message - сообщение для поставщика
Возвращает:
OfferId - идентификатор предложения по которому была попытка сделать заказа
Posted - флаг указывает был ли заказ отправлен

end - окончание периода за который выводятся накладные

Id - идентификатор файла
